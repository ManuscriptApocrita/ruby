Менеджер паролей (редакция sqlite)

Функции программы:

✔ доступ к программе по мастер-ключу (eas33)
✔ создание паролей двух типов (случайная генерация и мнемоническая) 
✔ сохранение результата генерации с логином и комментарием, вводимым от пользователя, в sqlite базу (зашифрованный вид) 
✔ ручное добавление связки логин - пароль - комментарий в базу данных 
✔ просмотр существующих записей в базе данных с возможностью удаления 
✔ предупреждение о прохождении 3-х месячного периода существования пароля и напоминание о необходимости его смены
✔ перегенерация метода шифрования происходит ежемесячно
✔ в этой версии предусмотрено сохранение всех данных в формат JSON (хранится в архиве, папка data) при смене ключа шифрования (экспериментально)
✔ введеная функция проверки пароля на стойкость к взлому!

Примечание:
- для работы программы необходима поставляемая с ней папка "data" в той же директории, откуда запускается программа
- диапазон выбора длины пароля случайного типа от 6 до 35, мнемонического от 2 до 15 слов
- к любому мнемоническому паролю в конец добавляется "3!6" для большей вероятности прохождения проверки на допустимость
- в мнемонической генерации пароль образуется, опираясь на первые две буквы каждого слова из сгенерированной группы слов.
Первая записывается как малый регистр, вторая - большой
- для корректной работы программы изменять вручную данные sqlite базы настоятельно не рекомендуется
- русский язык шифрованию в данной версии не поддается
