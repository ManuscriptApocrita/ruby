Функции программы:

✔ доступ к программе по мастер-ключу (eas11)
✔ создание паролей двух типов (случайная генерация и мнемоническая) 
✔ сохранение результата генерации с логином, вводимым от пользователя, в xml базу (зашифрованный вид) 
✔ ручное добавление связки логин - пароль в базу данных 
✔ просмотр существующих записей в базе данных с возможностью удаления 
✔ предупреждение о прохождении 3-х месячного периода существования пароля и указание на необходимость его смены
✔ перегенерация метода шифрования происходит ежемесячно (необходимо ежегодно производить сброс счетчика командой update в главном меню)

Примечание:
- для работы программы необходима поставляемая с ней папка "data" в той же директории, откуда запускается программа
- диапазон выбора длины пароля случайного типа от 5 до 35, мнемонического от 2 до 15 слов
- к любому мнемоническому паролю в конец добавляется "3!6" для большей вероятности прохождения проверки на допустимость
- в мнемонической генерации пароль образуется, опираясь на первые две буквы каждого слова из сгенерированной группы слов. Первая записывается как малый регистр, вторая - большой
- русский язык шифрованию в данной версии не поддается
